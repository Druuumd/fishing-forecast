# Terms of Use (MVP)

Дата обновления: 2026-04-21

## 1) Назначение сервиса
Fishing Forecast MVP предоставляет оценочный прогноз клева и возможность вести журнал уловов.

## 2) Ограничение ответственности
- Прогноз носит информационный характер.
- Сервис не гарантирует фактический результат рыбалки.
- Пользователь самостоятельно принимает решения на основе прогноза.

## 3) Правила использования
Пользователь обязуется:
- не отправлять вредоносные или заведомо ложные данные;
- не обходить механизмы защиты API (rate-limit, auth);
- не использовать сервис для злоупотреблений.

## 4) Блокировка доступа
Владелец сервиса может ограничить доступ при:
- многократных попытках abuse,
- нарушении правил использования,
- угрозе безопасности сервиса.

## 5) Данные пользователя
Обработка данных выполняется по Privacy Policy:
- `docs/legal/privacy-policy-mvp.md`.

## 6) Изменение условий
Условия могут обновляться. Актуальная версия публикуется с датой обновления.

## 7) Контакты
По вопросам условий использования и претензий:
- Email публикуется через `GET /v1/legal/info` (`support_email`).
- Перед публичным релизом обязателен production email.
