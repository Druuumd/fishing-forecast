# Реестр рисков MVP

| ID | Риск | Вероятность | Влияние | Митигация | Владелец |
|---|---|---|---|---|---|
| R1 | Изменение HTML у источника температуры воды | Средняя | Высокое | Fallback-источник, ручной override, алерты | Data |
| R2 | Недостаток пользовательских данных для retrain | Высокая | Среднее | Conservative scoring, сбор baseline-данных, отложенный retrain | ML |
| R3 | Перегрузка одного разработчика | Высокая | Высокое | Scope freeze, release gates, отказ от Post-MVP задач | Product/Tech |
| R4 | Злоупотребление `POST /catch` | Средняя | Высокое | Rate limit, anti-spam, мониторинг аномалий | Backend |
| R5 | Потеря данных при сбое сервера | Низкая | Критичное | Backup/restore policy, drill, проверка RPO/RTO | Ops |
